# Phase C — Prechecks

This folder contains **optional** docs that help keep Phase C governance-clean by:

- summarizing which prerequisites are already certified upstream (Phase B / Phase 0C / Entry Gate),
- specifying what Phase C must *import and reference* rather than re-prove.

Recommended workflow:
- Maintain an “upstream coverage” doc per evidence run id.
- Reference it from Phase C run transcript docs under `docs/phase-group-c/phase_C/RUNS/<run_id>.md`.
