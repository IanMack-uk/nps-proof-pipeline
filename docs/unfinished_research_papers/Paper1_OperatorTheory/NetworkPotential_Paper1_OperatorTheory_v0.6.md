**Mathematical Scope**

This paper develops an operator-theoretic framework for structural interaction on weighted graphs based on admissible coupling operators and their integrable realisations through relational potentials.

Relational potentials generate self-adjoint Hessian operators whose equilibrium evaluations determine admissible coupling operators governing structural interaction in the network. The coupling operator induces an intrinsic interaction topology on the edge set through its sparsity structure. Operator-theoretic analysis on this topology yields a quantitative theory of structural propagation and equilibrium response.

The paper establishes:

- admissible coupling operators as the fundamental structural operators of relational networks,

- coupling operators generated by equilibrium Hessians of relational potentials,

- incident and nodewise operator decompositions,

- the edgeâ€“interaction graph induced by coupling-operator locality,

- spectral representations of structural modes,

- susceptibility operators describing equilibrium response,

- propagation semigroups generated by coupling operators,

- locality-induced propagation bounds on the edgeâ€“interaction graph,

- echo operators as scale-resolved probes of structural coupling,

- operator stability properties near equilibrium.

A central result establishes that structural locality induces quantitative bounds on the propagation of structural influence. These bounds yield an effective finite-speed propagation principle on the edgeâ€“interaction graph, under which structural perturbations propagate only across bounded interaction distance at finite propagation scale.

The resulting framework establishes a network operator theory in which structural topology emerges from the coupling operator governing structural interaction rather than from the combinatorial structure of the underlying graph.

Relational potentials provide canonical generating functionals whose equilibrium Hessians produce admissible coupling operators.

# 0 Introduction

## 0.1 Relational Potentials as Operator Systems

Admissible coupling operators on weighted graphs admit integrable realisations through relational potentials. In this formulation structural properties of equilibrium configurations are governed by linear operators derived from the second variation of the potential.

Let

``` math
G = (V,E)
```

be a finite undirected graph and let

``` math
\Omega_{G} \subset \mathbb{R}^{\mid E \mid}
```

denote a compact domain of admissible edge weights. A relational potential is a scalar functional

``` math
\Phi_{G}:\Omega_{G} \rightarrow \mathbb{R}
```

whose equilibrium configurations satisfy

``` math
\nabla\Phi_{G}(w^{*}) = 0.
```

Mathematical objects and notation follow the *Network Potential Mathematical Specification v1.0.0.*

Paper 0 established an admissible class of relational potentials characterised by structural locality and structural coordinate representation. Within that framework equilibrium configurations arise as stationary points of scalar functionals defined on the space of edge-weight configurations.

The present paper develops the operator-theoretic structure induced by relational potentials. In particular, relational potentials generate self-adjoint Hessian operators whose equilibrium evaluations determine admissible coupling operators governing structural interaction in the network.

The principal objects of study are the coupling operators and associated propagation operators derived from equilibrium configurations, together with the spectral and propagation properties that these operators determine. Relational potentials provide canonical generating functionals whose equilibrium Hessians produce admissible operators.

In particular, the coupling operator induces a natural interaction topology on the edge set through its sparsity structure. This interaction topology determines how structural influence propagates through the network and provides the geometric foundation for the operator theory developed in this paper.

## 0.2 Equilibrium Operators

Let

``` math
L
```

be the coupling operator associated with an interior equilibrium configuration. The coupling operator provides the fundamental linear representation of structural coupling within the network and serves as the generator for the propagation operators introduced later in this paper.

When the coupling operator arises from a relational potential $`\Phi_{G}`$, it may be written in the form

``` math
L = - H^{*},
```

where

``` math
H(w) = D^{2}\Phi_{G}(w)
```

denotes the Hessian operator of the relational potential and

``` math
H^{*} = H(w^{*})
```

denotes the equilibrium Hessian at an interior equilibrium configuration $`w^{*}`$.

The equilibrium Hessian governs the local curvature of the relational potential and determines stability properties of equilibrium configurations. The second variation of the potential at equilibrium satisfies

``` math
\delta^{2}\Phi_{G}(w^{*})\lbrack v\rbrack = v^{\top}H^{\text{\textbackslash*}}v
```

for perturbation directions

``` math
v \in \mathbb{R}^{\mid E \mid}.
```

At interior strict local maxima the coupling operator satisfies

``` math
L \succ 0.
```

The coupling operator therefore provides the canonical linear representation of structural interaction governing equilibrium response and propagation behaviour in the network.

## 0.3 Response vs Propagation

Local response of equilibrium configurations to small perturbations is governed by the susceptibility operator

``` math
S = (H^{*})^{- 1}.
```

The susceptibility operator determines first-order changes in equilibrium edge weights induced by small external perturbations. In particular, for perturbations

``` math
f \in \mathbb{R}^{\mid E \mid}
```

equilibrium displacements satisfy

``` math
H^{*}\delta w = f
```

with solution

``` math
\delta w = Sf.
```

This operator provides a linear-response description of equilibrium structure and forms the basis of comparative-statics analysis for relational potentials.

However, the susceptibility operator describes only the steady-state displacement induced by a perturbation and does not capture the intermediate propagation of structural influence through the network.

The operator-theoretic framework developed in this paper extends this static viewpoint by introducing propagation operators generated by the coupling operator $`L`$. These operators describe the transmission of infinitesimal perturbations through the interaction topology induced by the equilibrium Hessian.

## 0.4 Main Theorem â€” Locality-Induced Propagation Geometry

Structural locality of relational potentials implies that the equilibrium Hessian operator is sparse, so that structural coupling between edges is confined to finite structural neighbourhoods. The sparsity structure of the Hessian induces an edgeâ€“interaction graph on the edge set, and propagation operators generated by the coupling operator act locally on this graph.

Structural locality therefore induces an intrinsic metric propagation geometry on weighted networks in which structural influence spreads along paths of the edgeâ€“interaction graph and decays with interaction distance. This geometry is determined entirely by the interaction structure of the coupling operator and is independent of any particular representation of the relational potential.

The central result of this paper shows that structural locality imposes quantitative constraints on propagation operators and induces an effective finite propagation velocity.

Structural locality therefore induces an intrinsic metric geometry on weighted networks with finite propagation velocity.

### Theorem 0.4.1 (Locality Propagation Principle)

Let $`\Phi_{G}`$be a relational potential satisfying the structural locality assumptions of Paper 0, and let $`H^{*}`$denote the equilibrium Hessian at a hyperbolic interior equilibrium configuration.

Let

``` math
L = - H^{*}
```

be the associated coupling operator and let

``` math
U(t) = e^{- tL}
```

denote the propagation operators.

Let $`\mathcal{G}_{E}`$denote the edgeâ€“interaction graph induced by the sparsity structure of $`H^{*}`$, and let $`d(e,f)`$denote interaction distance on $`\mathcal{G}_{E}`$.

Then propagation operators satisfy exponential locality bounds of the form

``` math
\mid U(t)_{ef} \mid \leq Ce^{vt}\alpha^{d(e,f)}
```

for constants $`C > 0`$, $`v > 0`$, and $`0 < \alpha < 1`$depending only on the locality parameters of the relational potential.

Consequently, structural influence propagates through the edgeâ€“interaction graph with an effective finite propagation speed determined by the locality parameters.

The detailed proof of Theorem 0.4.1 is given in Section 6.2.

### Conceptual Consequence

Structural locality therefore induces an intrinsic metric propagation structure on weighted networks. The interaction distance on the edgeâ€“interaction graph acts as the natural geometric scale governing transmission of structural influence, while the locality parameters determine a finite propagation velocity.

This result establishes that admissible coupling operators induce an intrinsic propagation geometry on weighted networks determined by the interaction structure of the coupling operator.

# 1 Operator Framework

This section establishes the operator framework used throughout the paper. Relational potentials induce linear operators acting on perturbations of edge-weight configurations, and a precise specification of configuration spaces and operator domains is required in order to develop spectral and propagation theory.

Throughout the paper let

``` math
G = (V,E)
```

be a finite undirected graph and let

``` math
\Omega_{G} \subset \mathbb{R}^{\mid E \mid}
```

denote a compact domain of admissible edge weights. Mathematical objects and notation follow the **Network Potential Mathematical Specification v1.0.0**.

Equilibrium configurations are denoted by

``` math
w^{*} \in \Omega_{G}.
```

Unless otherwise stated, equilibria considered in this paper are interior points of the admissible domain.

------------------------------------------------------------------------

## 1.1 Configuration Space

Edge-weight configurations are represented by vectors

``` math
w = (w_{e})_{e \in E}
```

in the admissible domain

``` math
\Omega_{G} \subset \mathbb{R}^{\mid E \mid}.
```

The ambient configuration space is therefore the finite-dimensional Euclidean space

``` math
\mathbb{R}^{\mid E \mid}.
```

Relational potentials are scalar functionals

``` math
\Phi_{G}:\Omega_{G} \rightarrow \mathbb{R}
```

that are twice continuously differentiable on the interior of the admissible domain.

Perturbations of equilibrium configurations are represented by vectors

``` math
v \in \mathbb{R}^{\mid E \mid}.
```

These perturbations describe infinitesimal variations of edge weights and form the basic objects on which the operators of this paper act.

------------------------------------------------------------------------

## 1.2 Tangent Spaces

For each configuration $`w \in \Omega_{G}`$the admissible perturbations are described by the tangent space

``` math
T_{w}\Omega_{G}.
```

If $`w`$lies in the interior of the admissible domain, then the tangent space coincides with the ambient space:

``` math
T_{w}\Omega_{G} = \mathbb{R}^{\mid E \mid}.
```

In particular, for interior equilibrium configurations $`w^{*}`$,

``` math
T_{w^{*}}\Omega_{G} = \mathbb{R}^{\mid E \mid}.
```

The restriction to interior equilibria allows linear operators derived from the relational potential to act on the full configuration space without feasibility constraints.

Tangent-space restrictions become essential when equilibria lie on the boundary of the admissible domain, a situation treated in Paper 2.

------------------------------------------------------------------------

## 1.3 Inner Product Structure

The configuration space is equipped with the standard Euclidean inner product

``` math
\langle u,v\rangle = u^{\top}v
```

for vectors $`u,v \in \mathbb{R}^{\mid E \mid}`$.

The associated norm is

``` math
\parallel v \parallel = \sqrt{v^{\top}v}.
```

All operators considered in this paper act on the finite-dimensional Hilbert space

``` math
\mathbb{R}^{\mid E \mid}
```

equipped with this inner product.

With respect to this inner product the Hessian operator

``` math
H(w)
```

is self-adjoint:

``` math
\langle u,H(w)v\rangle = \langle H(w)u,v\rangle.
```

Consequently the equilibrium Hessian

``` math
H^{*}
```

and the coupling operator

``` math
L = - H^{*}
```

are also self-adjoint operators.

This self-adjoint structure ensures that spectral decompositions and operator functions are well defined.

------------------------------------------------------------------------

## 1.4 Operator Domains

Operators induced by relational potentials act on perturbations of equilibrium configurations.

For an interior equilibrium $`w^{*}`$, the Hessian operator defines a linear map

``` math
H^{*}:T_{w^{*}}\Omega_{G} \rightarrow T_{w^{*}}\Omega_{G}.
```

Since

``` math
T_{w^{*}}\Omega_{G} = \mathbb{R}^{\mid E \mid},
```

the equilibrium Hessian acts on the entire configuration space:

``` math
H^{*}:\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}.
```

The coupling operator

``` math
L = - H^{*}
```

therefore defines a self-adjoint linear operator on the same space.

Propagation operators defined by

``` math
U(t) = e^{- tL}
```

act on the same domain:

``` math
U(t):\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}.
```

Similarly, the susceptibility operator

``` math
S = (H^{*})^{- 1}
```

acts as a linear operator on perturbations whenever the equilibrium Hessian is nonsingular.

The operators introduced in this paper therefore form a family of self-adjoint linear operators acting on the common configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

This shared operator domain allows spectral and propagation properties of relational potentials to be analysed within a unified operator framework.

# 2 Differential Operators

Relational potentials induce differential operators on the configuration space through their first and second derivatives.

The operator-theoretic analysis developed in this paper applies more generally to abstract coupling operators satisfying the locality and coercivity conditions introduced in Section 2.6. Relational potentials provide integrable realisations of these operators.

These operators describe marginal structural incentives and structural coupling between edges.

Let

``` math
\Phi_{G}:\Omega_{G} \rightarrow \mathbb{R}
```

be a relational potential satisfying the admissibility conditions established in Paper 0.

Throughout this section configurations are assumed to lie in the interior of the admissible domain unless otherwise stated.

The differential operators introduced here form the fundamental operator objects of relational potential theory.

------------------------------------------------------------------------

## 2.1 Marginal Structural Field

The first derivative of a relational potential defines a vector field on the configuration space.

### Definition 2.1.1 (Marginal Structural Field)

The **marginal structural field** associated with a relational potential $`\Phi_{G}`$is the vector-valued function

``` math
F(w) = \nabla\Phi_{G}(w)
```

with components

``` math
F_{e}(w) = \frac{\partial\Phi_{G}}{\partial w_{e}}(w),e \in E.
```

The marginal structural field represents the system of marginal structural incentives associated with edge weights.

For perturbations $`v \in \mathbb{R}^{\mid E \mid}`$, the first variation satisfies

``` math
D\Phi_{G}(w)\lbrack v\rbrack = \langle F(w),v\rangle.
```

Equilibrium configurations satisfy the stationarity condition

``` math
F(w^{*}) = 0.
```

Thus equilibrium configurations correspond to zeros of the marginal structural field.

The marginal structural field defines a continuously differentiable vector field on the interior of the admissible domain.

------------------------------------------------------------------------

## 2.2 Hessian Operator

The second derivative of the relational potential defines the fundamental linear operator governing structural interaction.

### Definition 2.2.1 (Hessian Operator)

The **Hessian operator** of a relational potential is defined by

``` math
H(w) = D^{2}\Phi_{G}(w).
```

For perturbations $`u,v \in \mathbb{R}^{\mid E \mid}`$,

``` math
D^{2}\Phi_{G}(w)\lbrack u,v\rbrack = u^{\top}H(w)v.
```

The matrix elements of the Hessian operator are

``` math
H_{ef}(w) = \frac{\partial^{2}\Phi_{G}}{\partial w_{e}\partial w_{f}}(w).
```

The Hessian operator describes the sensitivity of marginal structural incentives to changes in edge weights:

``` math
\frac{\partial F_{e}}{\partial w_{f}}(w) = H_{ef}(w).
```

Thus the Hessian operator provides the linearisation of the marginal structural field:

``` math
DF(w) = H(w).
```

------------------------------------------------------------------------

### Proposition 2.2.1 (Self-Adjointness)

For all configurations $`w`$in the interior of the admissible domain,

``` math
H(w)
```

is a self-adjoint operator on $`\mathbb{R}^{\mid E \mid}`$.

#### Proof

Since $`\Phi_{G}`$is twice continuously differentiable, mixed partial derivatives coincide:

``` math
\frac{\partial^{2}\Phi_{G}}{\partial w_{e}\partial w_{f}} = \frac{\partial^{2}\Phi_{G}}{\partial w_{f}\partial w_{e}}.
```

Thus

``` math
H_{ef}(w) = H_{fe}(w).
```

Therefore

``` math
H(w)
```

is symmetric and hence self-adjoint. â–ˇ

------------------------------------------------------------------------

## 2.3 Structural Locality

Structural locality, introduced axiomatically in Paper 0, imposes sparsity conditions on the Hessian operator.

### Definition 2.3.1 (Structural Neighbourhood)

For each edge $`e \in E`$, the **structural neighbourhood**

``` math
N(e) \subseteq E
```

is the minimal set of edges on which the marginal structural function $`F_{e}(w)`$depends.

Thus

``` math
F_{e}(w) = F_{e}(w_{N(e)}).
```

------------------------------------------------------------------------

### Proposition 2.3.1 (Locality of the Hessian)

If $`f \notin N(e)`$, then

``` math
H_{ef}(w) = 0.
```

#### Proof

Structural locality implies that $`F_{e}(w)`$depends only on weights $`w_{N(e)}`$. Therefore

``` math
\frac{\partial F_{e}}{\partial w_{f}}(w) = 0
```

whenever $`f \notin N(e)`$.

Since

``` math
H_{ef}(w) = \frac{\partial F_{e}}{\partial w_{f}}(w),
```

the result follows. â–ˇ

------------------------------------------------------------------------

### Structural Interpretation

Structural locality implies that the Hessian operator is sparse and that structural coupling between edges is local.

This locality property plays a fundamental role in the spectral and propagation results developed later in this paper.

In particular, locality ensures that propagation operators generated by the Hessian admit spatial decay properties analogous to those observed in local interaction systems.

------------------------------------------------------------------------

## 2.4 Equilibrium Hessian

Operator-theoretic analysis in this paper is centred on equilibrium configurations.

### Definition 2.4.1 (Equilibrium Hessian)

Let $`w^{*}`$be an equilibrium configuration. The **equilibrium Hessian** is defined by

``` math
H^{*} = H(w^{*}).
```

The equilibrium Hessian governs second-order behaviour of the relational potential near equilibrium.

For perturbations $`v`$,

``` math
\delta^{2}\Phi_{G}(w^{*})\lbrack v\rbrack = v^{\top}H^{*}v.
```

------------------------------------------------------------------------

### Definition 2.4.2 (Hyperbolic Equilibrium)

An equilibrium configuration $`w^{\text{\textbackslash*}}`$is **hyperbolic** if the equilibrium Hessian

``` math
H^{\text{\textbackslash*}}
```

is **nonsingular**.

Equivalently,

``` math
\ker(H^{\text{\textbackslash*}}) = \{ 0\}.
```

Hyperbolicity ensures that the equilibrium Hessian is invertible and therefore that the susceptibility operator is well defined.

------------------------------------------------------------------------

## 2.5 Coupling Operator

The Hessian operator determines structural coupling between edges.

### Definition 2.5.1 (Coupling Operator)

The **coupling operator** associated with an equilibrium configuration is defined by

``` math
L = - H^{*}.
```

The coupling operator provides the fundamental linear representation of structural interaction within the network.

------------------------------------------------------------------------

### Proposition 2.5.1 (Positive Definiteness)

If $`w^{*}`$is a strict local maximiser of the relational potential, then

``` math
L \succ 0.
```

#### Proof

Strict local maximality implies

``` math
v^{\top}H^{*}v < 0
```

for all nonzero perturbations $`v`$.

Therefore

``` math
v^{\top}Lv = - v^{\top}H^{*}v > 0
```

for all $`v \neq 0`$.

Thus $`L`$is positive definite. â–ˇ

------------------------------------------------------------------------

### Structural Interpretation

The coupling operator describes the linear propagation of structural influence between edges.

Diagonal elements of $`L`$describe intrinsic curvature of individual ties, while off-diagonal elements describe cross-edge coupling.

The coupling operator serves as the generator for propagation operators introduced in Section 6 and echo operators introduced in Section 7.

In the subsequent analysis the coupling operator will be treated as a primary structural object. Relational potentials provide a canonical mechanism for generating admissible coupling operators at equilibrium configurations.

## 2.6 Operator-First Formulation

The preceding sections introduced differential operators generated by relational potentials. In the operator-theoretic developments that follow, the coupling operator will be treated as a primary structural object independent of any particular potential representation.

This section introduces an abstract class of coupling operators for which the propagation and echo theories developed in later sections apply. Relational potentials provide canonical integrable realisations of operators within this class.

The admissible operator class represents the minimal class of operators for which susceptibility operators, propagation semigroups, and echo operators are well defined and satisfy quantitative locality bounds. Each assumption in Definition 2.6.1 corresponds to a structural requirement of the operator theory developed in subsequent sections:

- self-adjointness ensures real spectral decompositions and well-defined structural modes,

- coercivity guarantees invertibility and exponential propagation decay,

- structural locality induces the interaction geometry governing propagation,

- uniform boundedness ensures quantitative control of locality bounds.

Together these conditions define the natural class of local positive operators on weighted networks for which a consistent operator theory can be developed.

Let

``` math
G = (V,E)
```

be a finite undirected graph. The configuration space is identified with

``` math
\mathbb{R}^{\mid E \mid}
```

whose coordinates correspond to edge weights.

### Definition 2.6.1 (Admissible Coupling Operator)

A linear operator

``` math
L:\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}
```

is called an **admissible coupling operator** on $`G`$if the following conditions hold.

#### Self-adjointness

``` math
L = L^{\top}.
```

#### Coercivity

There exists a constant $`\sigma > 0`$such that

``` math
\langle v,Lv\rangle \geq \sigma \parallel v \parallel^{2}\text{for all }v \in \mathbb{R}^{\mid E \mid}.
```

Equivalently,

``` math
L \succ 0.
```

#### Structural Locality

For each edge $`e \in E`$define the interaction neighbourhood

``` math
\mathcal{N(}e) = \{\text{â€‰}f \in E:\text{\:\,}f \neq e\text{ and }L_{ef} \neq 0\text{â€‰}\}.
```

The operator is said to satisfy structural locality if

``` math
\Delta = \underset{e \in E}{\max} \mid \mathcal{N(}e) \mid < \infty.
```

Equivalently,

``` math
L_{ef} = 0\text{whenever}f \notin \mathcal{N(}e) \cup \{ e\}.
```

#### Uniform Local Boundedness

There exists a constant $`M > 0`$such that

``` math
\mid L_{ef} \mid \leq M\text{for all }e \neq f.
```

The interaction neighbourhoods define an undirected graph

``` math
\mathcal{G}_{E} = (E\mathcal{,I),}
```

called the **edgeâ€“interaction graph**, where

``` math
(e,f\mathcal{) \in I}\text{if and only if}e \neq f\text{ and }L_{ef} \neq 0.
```

The maximum interaction degree

``` math
\Delta = \underset{e \in E}{\max} \mid \mathcal{N(}e) \mid 
```

is called the **interaction degree** of the operator.

Admissible coupling operators describe systems in which structural interactions are local, uniformly bounded, and coercive. These properties ensure that susceptibility operators, propagation operators, and echo operators are well defined and admit quantitative locality bounds.

## Structural Interpretation

An admissible coupling operator specifies a system of linear structural interactions between edges.

Diagonal entries of $`L`$describe intrinsic curvature of individual ties, while off-diagonal entries describe pairwise structural coupling between edges within the interaction neighbourhoods.

The edgeâ€“interaction graph $`\mathcal{G}_{E}`$represents the structural interaction geometry induced by the operator. Distances in $`\mathcal{G}_{E}`$determine the propagation geometry governing the evolution operators introduced in Section 6.

### Proposition 2.6.1 (Operators Generated by Relational Potentials)

Let $`\Phi_{G}`$be an admissible relational potential in the sense of Paper 0, and let $`w^{\text{\textbackslash*}}`$be a hyperbolic strict local maximiser in the interior of the admissible domain.

Then the coupling operator

``` math
L = - D^{2}\Phi_{G}(w^{\text{\textbackslash*}})
```

is an admissible coupling operator.

#### Proof

Self-adjointness follows from Proposition 2.2.1.

Coercivity follows from Proposition 2.5.1.

Structural locality follows from Proposition 2.3.1, which implies that Hessian entries vanish outside structural neighbourhoods.

Uniform boundedness follows from continuity of the Hessian on compact neighbourhoods of $`w^{\text{\textbackslash*}}`$.

Therefore $`L`$satisfies Definition 2.6.1. â–ˇ

### Proposition 2.6.2 (Quadratic Realisation)

Let $`L`$be an admissible coupling operator and let

``` math
w^{\text{\textbackslash*}} \in \mathbb{R}^{\mid E \mid}.
```

The quadratic functional

``` math
\Phi_{L}(w) = - \frac{1}{2}\langle w - w^{\text{\textbackslash*}},\text{â€‰}L(w - w^{\text{\textbackslash*}})\rangle
```

satisfies

``` math
- D^{2}\Phi_{L}(w^{\text{\textbackslash*}}) = L.
```

Thus every admissible coupling operator admits a **local quadratic potential realisation** as the equilibrium Hessian of a relational potential.

Moreover, the quadratic functional satisfies the admissibility axioms of Paper 0 locally near $`w^{\text{\textbackslash*}}`$.

#### Proof

Differentiation yields

``` math
\nabla\Phi_{L}(w) = - L(w - w^{\text{\textbackslash*}}),
```

and therefore

``` math
D^{2}\Phi_{L}(w) = - L.
```

Evaluating at $`w^{\text{\textbackslash*}}`$gives

``` math
- D^{2}\Phi_{L}(w^{\text{\textbackslash*}}) = L.
```

Local admissibility follows from structural locality and boundedness of $`L`$. â–ˇ

#### Operator-Theoretic Formulation

The propagation and echo theories developed in later sections depend only on the coupling operator and its interaction geometry.

Accordingly, the principal results of this paper apply to admissible coupling operators independently of their origin.

Relational potentials therefore provide integrable realisations of admissible coupling operators, while the operator theory developed in this paper describes the structural behaviour common to all operators in this class.

# 3 Incident Operators

The differential operators introduced in Section 2 act on the configuration space

``` math
\mathbb{R}^{\mid E \mid}
```

without explicit reference to the node structure of the underlying graph. However, relational potentials are defined on weighted graphs, and structural interactions arise through nodewise incidence relations between edges.

This section introduces operator constructions that incorporate the incidence structure of the graph. These constructions provide a decomposition of the equilibrium Hessian into nodewise components and establish a link between global coupling operators and local structural interactions.

Incident operators provide a canonical local decomposition of the equilibrium Hessian. While the Hessian operator describes global structural coupling between ties, relational interactions arise through nodewise incidence relations in the underlying graph. The incident operators therefore identify the local structural contributions associated with individual nodes and provide a bridge between global operator structure and local network architecture.

Incident operators provide the canonical localisation of structural curvature to nodes and form the operator-theoretic basis for structural differentiation results developed in subsequent papers.

This decomposition plays a central role in the operator theory developed in later sections. In particular, incident operators determine the structure of nodewise perturbations and provide the natural probes used in the construction of echo operators in Section 7. Propagation locality established in Section 6 is reflected in the structure of these local operators.

Let

``` math
G = (V,E)
```

be a finite undirected graph and let

``` math
w^{*}
```

be an interior equilibrium configuration.

------------------------------------------------------------------------

## 3.1 Incident Hessian Operator

Structural coupling between edges arises through their shared incidence at nodes. The equilibrium Hessian may therefore be decomposed according to nodewise contributions.

For each node $`i \in V`$, define the **incident edge set**

``` math
E(i) = \{ e \in E:e\text{ is incident to }i\}.
```

Edges belonging to $`E(i)`$share a common node and therefore participate in local structural interactions.

------------------------------------------------------------------------

### Definition 3.1.1 (Incident Projection Operator)

For each node $`i \in V`$, define the projection operator

``` math
P_{i}:\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}
```

by

``` math
\left( P_{i}v)_{e}=\left\{ \begin{matrix}
v_{e} & e \in E(i) \\
0 & e \notin E(i).
\end{matrix} \right.\  \right.\ 
```

The operator $`P_{i}`$restricts perturbations to edges incident to node $`i`$.

Each projection operator is self-adjoint and idempotent:

``` math
P_{i}^{2} = P_{i},P_{i}^{\top} = P_{i}.
```

------------------------------------------------------------------------

### Definition 3.1.2 (Incident Hessian Operator)

For each node $`i \in V`$, the **incident Hessian operator** is defined by

``` math
H_{i} = P_{i}H^{*}P_{i}.
```

The operator $`H_{i}`$describes structural curvature restricted to perturbations supported on edges incident to node $`i`$.

------------------------------------------------------------------------

### Properties

Each incident Hessian operator is self-adjoint:

``` math
H_{i}^{\top} = H_{i}.
```

Furthermore,

``` math
H_{i}:Range(P_{i}) \rightarrow Range(P_{i}).
```

Thus each $`H_{i}`$defines a local curvature operator associated with node $`i`$.

------------------------------------------------------------------------

### Structural Interpretation

The incident Hessian operator represents the local curvature structure associated with a node. Its matrix elements describe pairwise structural interactions between ties incident to the same node.

These operators provide a nodewise decomposition of structural coupling that plays a central role in structural differentiation results developed in Paper I.

------------------------------------------------------------------------

## 3.2 Nodewise Coupling Operators

The coupling operator introduced in Section 2 admits a corresponding incident decomposition.

### Definition 3.2.1 (Nodewise Coupling Operator)

For each node $`i \in V`$, define the **nodewise coupling operator**

``` math
L_{i} = - H_{i}.
```

Thus

``` math
L_{i} = - P_{i}H^{*}P_{i}.
```

Each operator $`L_{i}`$describes local structural coupling between ties incident to node $`i`$.

------------------------------------------------------------------------

### Proposition 3.2.1 (Positive Semidefiniteness)

If $`w^{*}`$is a strict local maximiser, then each nodewise coupling operator satisfies

``` math
L_{i} \succeq 0.
```

#### Proof

For any perturbation $`v`$,

``` math
v^{\top}L_{i}v = - (P_{i}v)^{\top}H^{*}(P_{i}v).
```

Since strict local maximality implies

``` math
u^{\top}H^{*}u < 0
```

for all nonzero $`u`$, the result follows. â–ˇ

------------------------------------------------------------------------

### Structural Interpretation

The nodewise coupling operators measure the strength of structural interaction among ties incident to a given node.

Large eigenvalues of $`L_{i}`$correspond to strongly coupled local structures, while small eigenvalues correspond to weakly constrained local configurations.

These local operators provide a natural operator-theoretic representation of structural roles associated with nodes.

------------------------------------------------------------------------

## 3.3 Localâ€“Global Decomposition

The equilibrium Hessian may be interpreted as a global operator assembled from local structural interactions.

The projection operators satisfy

``` math
\sum_{i \in V}^{}P_{i} \neq I,
```

since each edge is incident to two nodes. Nevertheless, the incident operators provide a covering of the configuration space:

``` math
\bigcup_{i \in V}^{}{Range(}P_{i}) = \mathbb{R}^{\mid E \mid}.
```

------------------------------------------------------------------------

### Proposition 3.3.1 (Local Determination of Coupling)

If structural locality holds, then for each pair of edges $`e,f`$,

``` math
H_{ef}^{*} \neq 0
```

only if $`e`$and $`f`$belong to a common structural neighbourhood.

In particular, if edges share no structural neighbourhood, then

``` math
H_{ef}^{*} = 0.
```

------------------------------------------------------------------------

### Structural Operator Interpretation

The equilibrium Hessian therefore defines a sparse operator assembled from overlapping local components:

``` math
H^{*} = \text{global coupling operator assembled from local interactions}.
```

The incident operators $`H_{i}`$provide canonical local approximations to this global operator.

This localâ€“global structure allows structural properties of relational potentials to be analysed through nodewise operators while retaining global operator coherence.

# 4 Spectral Theory

The operator-theoretic structure induced by relational potentials is determined by the spectral properties of the equilibrium Hessian and the associated coupling operator.

Let $`w^{*}`$be an interior equilibrium configuration and let

``` math
H^{*} = H(w^{*})
```

denote the equilibrium Hessian operator.

Define the coupling operator

``` math
L = - H^{*}.
```

If $`w^{*}`$is a strict local maximiser, then $`L`$is positive definite and therefore admits a complete spectral decomposition.

The spectral properties of $`L`$determine:

- structural rigidity,

- propagation behaviour,

- response sensitivity,

- echo stability.

Throughout this section operators act on the configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

## 4.1 Edgeâ€“Interaction Graph

The coupling operator induces a natural interaction topology on the set of edges through the sparsity structure of the operator.

### Definition 4.1.1 (Edgeâ€“Interaction Graph)

Let

``` math
L
```

be the coupling operator associated with an equilibrium configuration. The **edgeâ€“interaction graph** is the undirected graph

``` math
G_{E} = (E\mathcal{,I)}
```

whose vertex set is the edge set $`E`$of the original network and whose interaction set $`\mathcal{I \subseteq}E \times E`$is defined by

``` math
(e,f\mathcal{) \in I}\text{if and only if}L_{ef} \neq 0.
```

Thus two edges interact whenever the coupling operator directly couples their weights.

The edgeâ€“interaction graph therefore represents the structural coupling topology induced by the coupling operator.

### Structural Neighbourhoods

For each edge $`e \in E`$, define the structural neighbourhood

``` math
\mathcal{N(}e) = \{\text{â€‰}f \in E:L_{ef} \neq 0\text{â€‰}\}.
```

When the coupling operator arises from a relational potential, the structural neighbourhood $`\mathcal{N(}e)`$coincides with the locality neighbourhood $`N(e)`$introduced in Paper 0 and Section 2.3. In the general operator-theoretic setting, structural neighbourhoods are determined directly by the sparsity structure of the coupling operator.

Structural neighbourhoods determine the set of edges that interact directly with edge $`e`$under the coupling operator.

#### Interaction Degree

Define the interaction degree

``` math
\Delta = \underset{e \in E}{\max} \mid \mathcal{N(}e) \mid .
```

The interaction degree measures the maximal number of edges directly coupled by the coupling operator.

This quantity plays a central role in the propagation bounds established in Section 6.

### Interaction Distance

The edgeâ€“interaction graph induces a natural metric on the edge set.

### Definition 4.1.2 (Interaction Distance)

For edges $`e,f \in E`$, the interaction distance

``` math
d(e,f)
```

is defined as the shortest-path distance between $`e`$and $`f`$in the edgeâ€“interaction graph $`G_{E}`$.

Interaction distance measures the minimal number of structural interactions required for perturbations to propagate between edges.

This metric provides the notion of distance used in the propagation bounds of Section 6 and the echo locality results of Section 7.

#### Interpretation

The edgeâ€“interaction graph represents the intrinsic topology of structural coupling induced by the coupling operator.

Unlike the original node graph

``` math
G = (V,E),
```

which represents adjacency of nodes, the edgeâ€“interaction graph represents adjacency of structural interactions.

Propagation and echo operators act locally on the edgeâ€“interaction graph rather than on the original node graph.

Thus, interaction distance emerges from the operator-theoretic structure of structural coupling rather than from the combinatorial structure of the underlying graph.

## 4.2 Spectral Decomposition

Since the coupling operator $`L`$is self-adjoint, the spectral theorem applies.

### Theorem 4.2.1 (Spectral Decomposition)

Let $`L`$be the coupling operator associated with a hyperbolic equilibrium configuration. Then there exist orthonormal eigenvectors

``` math
\left\{ v_{k}\}_{k = 1}^{\mid E \mid} \right.\ 
```

and eigenvalues

``` math
\lambda_{k} > 0
```

such that

``` math
Lv_{k} = \lambda_{k}v_{k}
```

and

``` math
L = \sum_{k = 1}^{\mid E \mid}\lambda_{k}v_{k}v_{k}^{\top}.
```

The eigenvectors form an orthonormal basis of the configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

#### Spectral Representation of Perturbations

Any perturbation vector

``` math
v \in \mathbb{R}^{\mid E \mid}
```

admits the expansion

``` math
v = \sum_{k}^{}c_{k}v_{k}
```

with coefficients

``` math
c_{k} = \langle v,v_{k}\rangle.
```

The quadratic form associated with the coupling operator satisfies

``` math
v^{\top}Lv = \sum_{k}^{}\lambda_{k}c_{k}^{2}.
```

Thus eigenvalues measure curvature strength along structural directions determined by the eigenvectors.

Large eigenvalues correspond to directions of strong curvature, while small eigenvalues correspond to directions of weak curvature.

These spectral quantities govern structural rigidity and propagation behaviour in later sections.

## 4.3 Structural Modes

Eigenvectors of the coupling operator define canonical structural directions near equilibrium.

### Definition 4.3.1 (Structural Modes)

Eigenvectors $`v_{k}`$of the coupling operator $`L`$are called **structural modes**.

Each structural mode represents a coordinated perturbation pattern across edge weights.

Structural modes describe intrinsic directions of structural variation in the neighbourhood of equilibrium.

#### Interpretation

Structural modes may be interpreted as the fundamental deformation patterns of the relational architecture.

Large eigenvalues correspond to directions of strong structural resistance:

``` math
\lambda_{k}\text{ large} \Rightarrow \text{rigid structural mode}.
```

Small eigenvalues correspond to directions of weak structural resistance:

``` math
\lambda_{k}\text{ small} \Rightarrow \text{flexible structural mode}.
```

Flexible modes represent directions along which structural reconfiguration occurs most easily.

These modes play a central role in the propagation operators of Section 6 and the echo operators of Section 7.

#### Relation to Structural Differentiation

Structural differentiation results developed in Paper I may be interpreted in terms of spectral separation between structural modes.

Strong spectral separation produces stable differentiation between structural roles, while small spectral gaps correspond to structurally flexible regimes.

## 4.4 Spectral Bounds

Spectral bounds control stability and propagation behaviour.

Let

``` math
0 < \lambda_{\min} \leq \lambda_{k} \leq \lambda_{\max}
```

denote the extreme eigenvalues of the coupling operator $`L`$.

### Proposition 4.4.1 (Quadratic Bounds)

For all perturbations

``` math
v \in \mathbb{R}^{\mid E \mid}
```

the quadratic form satisfies

``` math
\lambda_{\min} \parallel v \parallel^{2} \leq v^{\top}Lv \leq \lambda_{\max} \parallel v \parallel^{2}.
```

#### Proof

Using the spectral expansion

``` math
v = \sum_{k}^{}c_{k}v_{k}
```

we obtain

``` math
v^{\top}Lv = \sum_{k}^{}\lambda_{k}c_{k}^{2}.
```

Since

``` math
\lambda_{\min} \leq \lambda_{k} \leq \lambda_{\max},
```

we obtain

``` math
\lambda_{\min}\sum_{k}^{}c_{k}^{2} \leq \sum_{k}^{}\lambda_{k}c_{k}^{2} \leq \lambda_{\max}\sum_{k}^{}c_{k}^{2}.
```

Since

``` math
\sum_{k}^{}c_{k}^{2} = \parallel v \parallel^{2},
```

the bounds follow.

# 5 Response Operators

Local response of equilibrium configurations to small perturbations is governed by the inverse of the equilibrium Hessian operator. This operator provides a linear mapping between perturbations of marginal structural incentives and the resulting equilibrium displacements.

Let

``` math
w^{*}
```

be an interior hyperbolic equilibrium configuration and let

``` math
H^{*} = H(w^{*})
```

denote the equilibrium Hessian.

Since hyperbolicity implies invertibility of $`H^{*}`$, the inverse operator is well defined.

Response operators describe the steady-state displacement induced by infinitesimal perturbations and provide the operator-theoretic formulation of comparative statics for relational potentials.

------------------------------------------------------------------------

## 5.1 Susceptibility Operator

### Definition 5.1.1 (Susceptibility Operator)

The **susceptibility operator** is defined by

``` math
S = (H^{*})^{- 1}.
```

The susceptibility operator is a self-adjoint linear operator acting on the configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

The operator maps perturbations of marginal structural incentives to equilibrium displacements.

------------------------------------------------------------------------

### Spectral Representation

Using the spectral decomposition of Section 4,

``` math
L = \sum_{k}^{}\lambda_{k}v_{k}v_{k}^{\top},
```

the susceptibility operator admits the representation

``` math
S = - \sum_{k}^{}\frac{1}{\lambda_{k}}v_{k}v_{k}^{\top}.
```

Thus each structural mode responds independently with gain proportional to $`1/\lambda_{k}`$.

------------------------------------------------------------------------

### Interpretation

Structural modes with small eigenvalues produce large responses:

``` math
\lambda_{k}\text{ small } \Rightarrow \text{high sensitivity}.
```

Structural modes with large eigenvalues produce small responses:

``` math
\lambda_{k}\text{ large } \Rightarrow \text{low sensitivity}.
```

# 6 Propagation Operators

The susceptibility operator introduced in Section 5 describes the steady-state displacement induced by infinitesimal perturbations. However, susceptibility does not describe the intermediate propagation of structural influence through the network.

Propagation operators provide a parameterised description of how perturbations spread through the network under the structural coupling determined by the equilibrium Hessian.

Let

``` math
w^{*}
```

be an interior hyperbolic equilibrium configuration and let

``` math
H^{*} = H(w^{*})
```

denote the equilibrium Hessian.

Define the coupling operator

``` math

```

If $`w^{*}`$is a strict local maximiser, then $`L`$is positive definite and self-adjoint.

The coupling operator therefore generates a family of propagation operators describing the transmission of infinitesimal perturbations through the network.

Propagation operators provide a continuous interpolation between the initial perturbation and its long-term dissipation.

In addition to their spectral properties, propagation operators satisfy quantitative locality bounds derived from the structural neighbourhood assumptions of Section 2.

These bounds are expressed in terms of the interaction topology introduced in Section 4.

## 6.1 Edgeâ€“Edge Locality and Interaction Distance

Structural locality implies that the coupling operator possesses a sparse interaction structure. This structure induces a natural notion of propagation distance between edges.

Let

``` math
w^{*}
```

be an interior hyperbolic equilibrium configuration and let

``` math
H^{*} = H(w^{*})
```

denote the equilibrium Hessian. Let

``` math
L = - H^{*}
```

be the associated coupling operator.

Structural interactions between edges are described by the edgeâ€“interaction graph

``` math

```

introduced in Section 4.1.

Propagation of perturbations occurs through chains of interactions on $`\mathcal{G}_{E}`$, so that influence propagates along paths in the edgeâ€“interaction graph.

Interaction distance is defined in **Definition 4.1.2** and will be used throughout this section.

### Locality Parameters

Assume the coupling operator satisfies uniform locality bounds.

#### Interaction Degree

The interaction degree is defined by

``` math
\Delta = \underset{e \in E}{\max} \mid \{ f \neq e:L_{ef} \neq 0\} \mid .
```

This quantity equals the maximal degree of a vertex in the edgeâ€“interaction graph $`\mathcal{G}_{E}`$.

### Local Coupling Bound

The local coupling bound is defined by

``` math
M = \underset{e \neq f}{\max} \mid L_{ef} \mid .
```

This quantity bounds the strength of direct structural coupling between distinct edges.

#### Interpretation

The parameters

``` math
\Delta\text{and}M
```

quantify structural locality at the operator level.

The interaction degree $`\Delta`$measures the maximal number of direct structural interactions associated with a single edge, while the coupling bound $`M`$measures the maximal strength of direct interaction.

Together these parameters determine the effective propagation velocity governing transmission of structural influence in Section 6.2.

## 6.2 Locality-Induced Propagation Bound

Structural locality implies that propagation operators transmit influence at a controlled rate across the interaction graph.

The following theorem establishes a quantitative propagation bound in terms of the interaction distance introduced in Definition 4.1.2.

Theorem 6.2.1 provides the proof of Theorem 0.4.1.

Structural influence propagates along paths in the edgeâ€“interaction graph $`\mathcal{G}_{E}`$, and the magnitude of propagation between edges is controlled by interaction distance and by the locality parameters defined in Section 6.1.

### Theorem 6.2.1 (Locality-Induced Propagation Bound)

Let

``` math
U(t) = e^{- tL}
```

be the propagation semigroup.

Let $`d(e,f)`$denote interaction distance (Definition 4.1.2).

Then for all edges $`e,f \in E`$and all $`t \geq 0`$,

``` math
\mid U(t)_{ef} \mid \leq \sum_{k \geq d(e,f)}^{}\frac{\left( t\Delta M)^{k} \right.\ }{k!}.
```

Writing

``` math
r = d(e,f),x = t\Delta M,
```

we obtain the bound

``` math
\mid U(t)_{ef} \mid \leq e^{x}\left( \frac{ex}{r} \right)^{r}.
```

Consequently there exists a propagation velocity scale

``` math
v = \Delta M
```

such that influence outside the region

``` math
d(e,f) \lesssim vt
```

decays exponentially with interaction distance.

#### Proof

Using the exponential series expansion,

``` math
U(t) = \sum_{k = 0}^{\infty}\frac{t^{k}}{k!}L^{k},
```

we obtain

``` math
U(t)_{ef} = \sum_{k = 0}^{\infty}\frac{t^{k}}{k!}(L^{k})_{ef}.
```

If $`k < d(e,f)`$, no sequence of structural couplings of length $`k`$connects $`e`$to $`f`$, and therefore

``` math
\left( L^{k})_{ef}=0. \right.\ 
```

Thus

``` math
\mid U(t)_{ef} \mid \leq \sum_{k \geq d(e,f)}^{}\frac{t^{k}}{k!} \mid (L^{k})_{ef} \mid .
```

Each factor satisfies

``` math
\mid L_{uv} \mid \leq M
```

and at most $`\Delta`$indices may be chosen at each step, giving

``` math
\mid (L^{k})_{ef} \mid \leq (\Delta M)^{k}.
```

Substitution yields

``` math
\mid U(t)_{ef} \mid \leq \sum_{k \geq d(e,f)}^{}\frac{\left( t\Delta M)^{k} \right.\ }{k!}.
```

The exponential-tail estimate

``` math
\sum_{k \geq r}^{}\frac{x^{k}}{k!} \leq e^{x}\left( \frac{ex}{r} \right)^{r}
```

gives the stated bound.

â–ˇ

#### Interpretation

Propagation operators satisfy an effective finite-speed propagation principle on the interaction graph.

Structural influence originating at edge $`f`$reaches edge $`e`$only after traversing interaction distance $`d(e,f)`$, and influence outside the region

``` math
d(e,f) \approx vt
```

is exponentially suppressed.

The parameter $`t`$therefore represents probing depth measured in structural couplings rather than physical time.

## 6.3 Propagation Semigroup

Propagation of perturbations is governed by the operator semigroup generated by the coupling operator.

### Definition 6.3.1 (Propagation Semigroup)

Propagation operators are defined by

``` math

```

Each operator

``` math
U(t):\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}
```

is self-adjoint and positive definite.

#### Semigroup Property

Propagation operators satisfy

``` math
U(0) = I
```

and

``` math
U(t + s) = U(t)U(s)
```

for all $`t,s \geq 0`$.

Thus $`U(t)`$forms a strongly continuous semigroup on the configuration space.

#### Differential Equation

Propagation operators solve the linear evolution equation

``` math
\frac{d}{dt}v(t) = - Lv(t),
```

with initial condition

``` math
v(0) = v_{0}.
```

The solution is

``` math
v(t) = U(t)v_{0}.
```

Propagation operators therefore describe diffusion of perturbations under the structural coupling operator.

## 6.4 Spectral Representation


be the spectral decomposition of Section 4.

### Theorem 6.4.1 (Spectral Representation)


Proof follows from the spectral calculus for self-adjoint operators.

â–ˇ

### Proposition 6.5.1 (Exponential Decay Bound)


### Theorem 6.6.1 (Integral Representation)

The susceptibility operator admits the representation


#### Interpretation

The susceptibility operator may therefore be interpreted as accumulated propagation response:

Propagation operators provide a scale-resolved decomposition of equilibrium response.

# 7 Echo Operators

Propagation operators introduced in Section 6 describe the forward transmission of perturbations through the network under the structural coupling operator. Echo operators extend this framework by introducing forwardâ€“perturbâ€“backward propagation processes that probe structural coupling in a scale-resolved manner.

Echo operators provide a parameterised family of structural probes derived from the coupling operator

``` math
L = - H^{*}.
```

Unlike susceptibility operators, which describe steady-state response, and propagation operators, which describe forward diffusion, echo operators measure the degree to which perturbations propagate through the network and return under reversed propagation.

Echo operators therefore provide a **finite-depth diagnostic of structural coupling** complementary to the steady-state response described by susceptibility operators.

The locality-induced propagation bound of Theorem 6.2.1 implies that echo operators act as localized structural probes whose effective range is determined by the propagation scale $`t`$.

Throughout this section operators act on the configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

# 7.1 Echo Operators and Echo Channels

Echo operators describe forward propagation followed by reversed propagation with an intermediate structural perturbation.

Let

``` math
U(t) = e^{- tL}
```

denote the propagation semigroup introduced in Section 6.

## Definition 7.1.1 (Echo Operator)

For propagation scale $`t \geq 0`$, define the echo operator

``` math
E(t) = U(t)PU(t).
```

Here

``` math
P:\mathbb{R}^{\mid E \mid} \rightarrow \mathbb{R}^{\mid E \mid}
```

is a bounded self-adjoint perturbation operator representing a structural probe.

The operator $`P`$specifies the structural perturbation or measurement, while the propagation operators $`U(t)`$determine the depth of structural probing.

### Structural Interpretation of Probe Operators

In the present framework probe operators typically represent local structural perturbations or structural measurements derived from the geometry of the equilibrium Hessian.

Important examples include:

- edge projection operators $`P_{e}`$,

- nodewise projection operators $`P_{i}`$introduced in Section 3,

- structural neighbourhood projections,

- incident-Hessian operators

``` math
H_{i} = P_{i}HP_{i}.
```

Such operators are naturally self-adjoint due to the symmetry of the Hessian operator

``` math
H^{*} = H(w^{*}),
```

ensuring that echo operators preserve the self-adjoint operator structure induced by relational potentials.

## Definition 7.1.2 (Echo Channels)

Let $`P_{e}`$denote the projection onto edge $`e`$. The **edge echo channel** is defined by

``` math
E_{e}(t) = U(t)P_{e}U(t).
```

Similarly, for nodewise projections $`P_{i}`$, the **node echo channel** is

``` math
E_{i}(t) = U(t)P_{i}U(t).
```

Echo channels describe how perturbations introduced at a structural location propagate through the interaction graph and return to that location.

### Interpretation

Small values of $`t`$probe local structural coupling.

Large values of $`t`$probe global structural coupling.

Thus $`t`$defines a **structural probing depth** determined by propagation through the coupling operator.

# 7.2 Echo Locality and Operator Structure

Echo operators inherit the locality structure of propagation operators.

In particular, Theorem 6.2.1 implies that propagation operators transmit influence only across bounded interaction distance on the edge-interaction graph. Echo operators therefore act as localized structural probes.

## 7.2.1 Echo Locality Bound

### Theorem 7.2.1 (Echo Locality Bound)

Let

``` math
E(t) = U(t)PU(t)
```

be an echo operator and suppose the probe operator $`P`$is supported on a set of edges

``` math
S \subseteq E.
```

Then for all edges $`e,f \in E`$,

``` math
(E(t))_{ef} = \sum_{g,h \in S}^{}{U(t})_{eg}P_{gh}U(t)_{hf}.
```

Consequently,

``` math
\mid (E(t))_{ef} \mid \leq \sum_{g,h \in S}^{}{\mid U(t})_{eg} \mid \mid P_{gh} \mid \mid U(t)_{hf} \mid .
```

Using the propagation bound of Theorem 6.2.1, there exist constants $`C_{t} > 0`$and $`\mu > 0`$such that

``` math
\mid (E(t))_{ef} \mid \leq C_{t}\text{â€‰} \parallel P \parallel \text{â€‰}\exp( - \mu d(e,S))\exp( - \mu d(f,S)).
```

Thus echo responses decay exponentially with interaction distance and scale proportionally with the probe strength $`\parallel P \parallel`$.

### Proof

From the definition

``` math
E(t) = U(t)PU(t)
```

we obtain

``` math
(E(t))_{ef} = \sum_{g,h \in E}^{}{U(t})_{eg}P_{gh}U(t)_{hf}.
```

If $`P`$is supported on $`S`$, then

``` math
P_{gh} = 0\text{unless}g,h \in S,
```

giving

``` math
(E(t))_{ef} = \sum_{g,h \in S}^{}{U(t})_{eg}P_{gh}U(t)_{hf}.
```

Taking absolute values yields

``` math
\mid (E(t))_{ef} \mid \leq \sum_{g,h \in S}^{}{\mid U(t})_{eg} \mid \mid P_{gh} \mid \mid U(t)_{hf} \mid .
```

Applying Theorem 6.2.1 to both propagation factors yields exponential decay in interaction distance. â–ˇ

## 7.2.2 Operator Properties

Echo operators inherit structural properties from propagation operators.

### Proposition 7.2.2 (Self-Adjointness)

If $`P`$is self-adjoint, then

``` math
E(t)
```

is self-adjoint.

**Proof**

Since

``` math
U(t)^{\top} = U(t)
```

and

``` math
P^{\top} = P,
```

we obtain

``` math
E(t)^{\top} = U(t)PU(t) = E(t).
```

â–ˇ

### Proposition 7.2.3 (Positivity)

If $`P`$is positive semidefinite, then

``` math
E(t)
```

is positive semidefinite.

**Proof**

For any perturbation $`v`$,

``` math
v^{\top}E(t)v = (U(t)v)^{\top}P(U(t)v).
```

Since $`P`$is positive semidefinite, the result follows.

â–ˇ

### Norm Bound

Echo operators satisfy

``` math
\parallel E(t) \parallel \leq \parallel P \parallel .
```

Thus echo operators remain bounded for all propagation scales.

### Spectral Representation

Using the spectral decomposition

``` math
U(t) = \sum_{k}^{}e^{- t\lambda_{k}}v_{k}v_{k}^{\top},
```

echo operators admit the representation

``` math
E(t) = \sum_{k,\mathcal{l}}^{}e^{- t(\lambda_{k} + \lambda_{\mathcal{l}})}v_{k}v_{k}^{\top}Pv_{\mathcal{l}}v_{\mathcal{l}}^{\top}.
```

This representation shows that echo operators couple structural modes through the probe operator.

# 7.3 Echo Sensitivity

Echo operators provide a quantitative measure of structural sensitivity.

## Definition 7.3.1 (Echo Sensitivity)

Let $`P`$be a projection operator. The echo sensitivity at scale $`t`$is defined by

``` math
\eta(t) = \parallel E(t) \parallel .
```

Echo sensitivity measures the maximal structural response detected by echo propagation.

### Spectral Bound

Using the exponential decay bound for propagation operators (Proposition 6.5.1), echo sensitivity satisfies

``` math
\eta(t) = \parallel E(t) \parallel \leq \parallel U(t) \parallel^{2} \parallel P \parallel \leq e^{- 2t\lambda_{\min}} \parallel P \parallel .
```

Thus, echo sensitivity decays exponentially with propagation scale.

### Interpretation

Large echo sensitivity indicates strong structural coupling between propagation paths.

Small echo sensitivity indicates weak propagation return.

Echo sensitivity therefore measures the degree to which perturbations propagate through the network and return to their origin.

# 7.4 Structural Scrambling

Echo operators provide a natural measure of structural scrambling.

Structural scrambling measures the extent to which perturbations disperse across structural modes during propagation.

## Definition 7.4.1 (Structural Scrambling Index)

Let $`P`$be a projection operator. The structural scrambling index is defined by

``` math
\chi(t) = tr(E(t)).
```

The scrambling index measures the total echo return across all structural modes.

### Locality Interpretation

The locality-induced propagation bound of Theorem 6.2.1 establishes a structural velocity scale

``` math
v = \Delta M.
```

This scale determines the rate at which structural influence spreads across the edge-interaction graph.

The scrambling index therefore measures the rate at which echo responses disperse relative to the locality-induced propagation scale.

Networks with large scrambling indices exhibit rapid dispersion of echo responses across interaction distance, while networks with small scrambling indices retain localized structural influence over larger propagation scales.

### Structural Interpretation

Echo operators distinguish between two regimes.

**Echo-Coherent Regime**

Perturbations propagate and return within a concentrated set of structural modes.

**Scrambling Regime**

Perturbations disperse across many structural modes.

These regimes correspond respectively to rigid and flexible structural configurations.

Echo operators therefore provide a propagation-based characterisation of structural architecture complementary to spectral rigidity measures.

# Structural Role of Echo Operators

Echo operators complete the operator-theoretic description of relational potentials by providing a scale-resolved characterisation of structural coupling.

Propagation operators describe the transmission of perturbations through the interaction graph, while susceptibility operators describe the accumulated equilibrium response. Echo operators interpolate between these regimes by resolving structural influence at finite propagation depth.

Together propagation and echo operators provide complementary descriptions of relational coupling:

``` math
{\text{propagation} \rightarrow \text{finite-depth structure},
}{\text{susceptibility} \rightarrow \text{equilibrium structure}.
}
```

Echo operators therefore provide the natural operator-theoretic framework for analysing structural interaction across multiple propagation scales.

## 7.5 Structural Determination by Echo Operators

Echo operators provide scale-resolved probes of structural coupling generated by the coupling operator $`L`$. The following result shows that echo operators determine the coupling operator uniquely when sufficiently rich probe families are available.

This establishes that echo operators are not merely diagnostic tools but structural invariants of admissible coupling operators.

### Definition 7.5.1 (Separating Probe Family)

A family of self-adjoint probe operators

``` math
\mathcal{P = \{}P_{\alpha}\}
```

is called **separating** if the linear span of the ranges of the operators $`P_{\alpha}`$is equal to the configuration space

``` math
\mathbb{R}^{\mid E \mid}.
```

Projection operators onto individual edges or nodes form separating probe families.

### Theorem 7.5.1 (Echo Reconstruction Theorem)

Let $`L_{1}`$and $`L_{2}`$be admissible coupling operators on a graph $`G`$with propagation semigroups

``` math
U_{i}(t) = e^{- tL_{i}},i = 1,2.
```

Let $`\mathcal{P}`$be a separating probe family and define echo operators

``` math
E_{i}(t) = U_{i}(t)PU_{i}(t),P \in \mathcal{P}.
```

If

``` math
E_{1}(t) = E_{2}(t)\text{for all }P \in \mathcal{P}\text{ and all }t \geq 0,
```

then

``` math
L_{1} = L_{2}.
```

Thus admissible coupling operators are uniquely determined by their echo operators.

#### Proof

For each probe operator $`P`$,

``` math
E_{i}(t) = U_{i}(t)PU_{i}(t).
```

Differentiating with respect to $`t`$at $`t = 0`$gives

``` math
\frac{d}{dt}E_{i}(t) \mid_{t = 0} = - L_{i}P - PL_{i}.
```

Equality of echo operators implies equality of derivatives:

``` math
L_{1}P + PL_{1} = L_{2}P + PL_{2}
```

for all probes $`P \in \mathcal{P}`$.

Since $`\mathcal{P}`$is separating, this identity determines the action of the operators on a spanning set of vectors.

Hence

``` math
L_{1} = L_{2}.
```

â–ˇ

### Interpretation

Theorem 7.5.1 shows that echo operators provide a complete structural characterisation of admissible coupling operators.

Propagation operators describe how perturbations spread through the interaction graph, while echo operators determine the underlying structural coupling that generates this propagation.

Echo operators therefore constitute structural invariants of admissible coupling operators.

## 8.3 Rigidityâ€“Propagation Principle


#### Proof

The susceptibility bound follows from spectral representation:

``` math

```

giving

``` math
\parallel S \parallel = \frac{1}{\lambda_{\min}} = \frac{1}{\sigma}.
```

``` math

```

implying

``` math
\parallel U(t) \parallel \leq e^{- t\sigma}.
```

``` math

```

and submultiplicativity of the operator norm:

``` math
\parallel E(t) \parallel \leq \parallel U(t) \parallel^{2} \parallel P \parallel \leq e^{- 2t\sigma} \parallel P \parallel .
```

â–ˇ

### Structural Interpretation

The rigidity parameter provides a single scalar measure controlling structural stability across all operator regimes.

Large rigidity corresponds to:

- small susceptibility

- rapid propagation decay

- rapid echo decay

Small rigidity corresponds to:

- large susceptibility

- slow propagation decay
